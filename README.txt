Hilda May Events — Static Website

Deploy Options:

1. Netlify Drop (Easiest)
   - Go to https://app.netlify.com/drop
   - Drag & drop index.html
   - Done (Netlify gives you a live URL)

2. GitHub Pages
   - Create a new GitHub repo
   - Upload index.html at the root
   - Repo → Settings → Pages → Deploy from branch → main
   - Your site will be live at username.github.io/repo-name
